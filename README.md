# FocusTarget

专注目标管理应用，帮助您跟踪和完成重要任务。

## 项目特点

- 使用Vue 3和TypeScript开发
- Tailwind CSS设计的现代UI
- 任务管理和专注模式
- PostgreSQL数据库支持

## 技术栈

- Vue 3 (Composition API)
- TypeScript
- Tailwind CSS
- Drizzle ORM
- PostgreSQL

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 数据库操作

```bash
# 生成数据库迁移文件
npm run db:generate

# 执行数据库迁移
npm run db:migrate

# 测试数据库连接
npm run db:test
```
